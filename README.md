# Pixl
**XML Parser**<br />
## How To Use
Include The Header file in your project.
```c
#include <Pixl.h>
```
Intialize a PixlDoc state.
```c
PixlDoc *doc = PixlDoc_Load("file.xml");

// Usage
// Will be Updated Soon
```
Close the state, to free the memory
```c
PixlDoc_Free(doc);
```
